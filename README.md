# Nirzo E-commerce App

Nirzo is a full-featured e-commerce application built with React Native. It provides a seamless shopping experience with features like product browsing, a dynamic cart, user authentication, and profile management.

### App Demo

   https://github.com/user-attachments/assets/c34fa030-13fb-4d5d-9d97-071437a52d30

## Features

- **Home Screen**: Browse featured products and categories.
- **Search**: Find products easily.
- **Product Details**: View detailed information, images, and pricing for individual products.
- **Shopping Cart**: Add products to cart, view total items via a dynamic badge in the bottom navigation.
- **User Authentication**: Login/Signup functionality.
- **Profile Management**: Manage user settings.
- **Responsive Navigation**: Bottom tab navigation fully optimized for modern devices (includes bottom safe area support for devices with notches or gesture navigation).

## Tech Stack

- **Framework**: React Native
- **Navigation**: React Navigation (Stack & Bottom Tabs)
- **State Management**: React Context API
- **Icons**: React Native Vector Icons (Ionicons)
- **UI/Layout**: React Native Safe Area Context

## Getting Started

### Prerequisites

- Node.js (v14 or newer)
- React Native environment setup (Android Studio / Xcode)

### Installation

1. Clone the repository and navigate to the project directory:
   ```bash
   cd Nirzo
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the application:
   - For Android:
     ```bash
     npm run android
     ```
   - For iOS:
     ```bash
     npm run ios
     ```

## Recent Updates
- Added dynamic safe area support for the bottom tab navigation to prevent overlap with the device's navigation bar on modern Android/iOS devices.
